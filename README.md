This project focuses on analyzing a Loan Approval dataset to understand the key factors influencing loan approval decisions.
The workflow includes data cleaning, handling missing values, exploratory data analysis, and data visualization using Python libraries.
Through statistical analysis and visual insights, the project highlights the impact of income, education, credit history, marital status, and property area on loan approval outcomes.
This project demonstrates a structured approach to data analysis and insight generation using real-world financial data.

Tools Used: Python, Pandas, NumPy, Matplotlib, Seaborn, Jupyter Notebook
